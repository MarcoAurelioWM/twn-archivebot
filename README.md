This bot archives discussion threads on [translatewiki.net](https://translatewiki.net) using [pywikibot](https://github.com/wikimedia/pywikibot) and
in particular [archivebot.py](https://github.com/wikimedia/pywikibot/scripts/archivebot.py). It operates through [Wikimedia Toolforge](https://wikitech.wikimedia.org/wiki/Portal:Toolforge).
